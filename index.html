Predictive Maintenance - Turbofan RUL

GitHub Live Demo

Predicts when jet engines will fail based on sensor data, before they actually break. Trained on NASA's C-MAPSS dataset (100 simulated turbofan engines, 21 sensors each). The interactive demo lets you step through an engine's life cycle by cycle and watch the predicted remaining life update in real time.

  • Feature engineering on raw sensor data: rolling averages, trends, rate of change
  • Two models compared: XGBoost (point estimates) and Survival Forest (probability curves)
  • SHAP explanations showing which sensors drive each prediction
  • Live demo with risk gauges that go from green to red as failure approaches

GitHub Live Demo

An in-browser associative analytics engine built on DuckDB-WASM. Supports cross-table filtering through auto-detected relationships - click any cell to see related data across all tables with visual state indicators. Includes a canvas mode for creating composite views from multiple tables.

  • Associative selection model: click-to-filter across related tables
  • Canvas mode with composite tables and automatic JOIN path detection
  • Fully client-side - DuckDB-WASM runs analytics in the browser
  • Visual state indicators: selected, possible, excluded
